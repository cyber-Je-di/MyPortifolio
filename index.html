My Work

MxM Accessories: Luxury E-commerce Platform

Developed a full-stack e-commerce platform for a luxury accessories brand using Flask and SQLite. The system manages a complete product lifecycle, from an Admin Dashboard for inventory and stock management to a dynamic customer-facing catalog with a shopping cart implementation. A key technical feature is the custom server-side image validation (using Python's Pillow library) which enforces a minimum 1000px resolution and applies high-quality compression to ensure a premium visual experience, directly addressing the brand's luxury positioning. The front-end leverages Jinja templating and GSAP for responsive, smooth user interactions.

The system includes integrated, role-specific dashboards, secure file uploads, and an internal messaging component, demonstrating a mastery of developing scalable, data-driven web applications.
